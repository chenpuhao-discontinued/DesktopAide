## DesktopAide

DeskAide是一款开源的基于Windows系统的桌面程序
它可以帮助您整理桌面，查找文件，尤其适合学校教师使用
### 注意
1. 如果您可以帮助我修改代码，请使用IntelliJ IDEA，谢谢！没有IDEA?[点我去下载](https://www.jetbrains.com/zh-cn/idea/download/)
2. 后续将会放弃Java开发前端，将使用C#~（只要作者想起来）~
